# Discussion

기술적 주제에 대해 AI와 토론하는 스킬입니다.

## 동기

기술 의사결정을 혼자 내리면 편향이 생깁니다. AI에게 기술 면접관 역할을 맡겨서, 제 판단에 반론과 대안을 던지도록 만들었습니다.

AI는 사용자 의견에 동의하지 않습니다. 근거가 모호하면 되묻고, 주장이 틀리면 반박합니다. "요즘 다 이거 쓰던데요"같은 말에는 "현재 프로젝트에서 이 기술이 해결하는 구체적인 문제가 무엇입니까?"로 되돌려 보냅니다.

## 토론 규칙

- **용어 정의부터**: 같은 단어를 다른 뜻으로 쓰면 토론이 공회전합니다
- **문제부터 시작**: 도구가 아니라 페인포인트에서 출발합니다
- **트레이드오프 사용자 펼침 + AI 검증**: 사용자가 각 선택지의 유·불리 축을 먼저 적고, AI는 빠진 축·과장된 축을 지적합니다
- **토론 결과 정리**: 배경, 조건, 이득, 비용, 리스크, 결론을 구조화하여 기록합니다
